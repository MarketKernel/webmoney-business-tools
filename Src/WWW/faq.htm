Главная::FAQ

Часто задаваемые вопросы по использованию программы WebMoney Business Tools

Зачем нужна программа?

Программа упрощает ведение бизнеса в системе WebMoney. Она содержит множество универсальных инструментов, которые решают наиболее распространенные задачи бизнеса.

Кроме этого, программа позволяет решать уникальные задачи, путем создания бизнес-автоматов.

На каких условиях распространяется программа?

Программа WebMoney Business Tools распространяется бесплатно с открытым исходным кодом по лицензии MIT.

Как работает программа?

Используются открытые XML-интерфейсы системы WebMoney.

Я скачал программу, а она не работает. Что делать?

Программа создана для современных операционных систем Windows Vista и более поздних. Для использования программы на более ранних системах, таких как Windows XP или Windows 2003, скачайте и установите дистрибутивный пакет Microsoft .NET Framework 2.0.

Является ли проект официальным сервисом WebMoney?

На данный момент нет.

Безопасно ли использовать программу?

Программа WebMoney Business Tools не имеет встроенных средств для противостояния вирусным атакам. Для ее безопасного использования, необходимо принятие специальных мер.

Где программа хранит данные?

  • Закрытый ключ WebMoney Keeper Clasic хранится в защищенном хранилище пользователя в зашифрованном виде.
  • Ключ WebMoney Keeper Light хранится в хранилище сертификатов пользователя (без возможности экспорта).
  • Настройки хранятся в рабочей папке программы в формате XML с применением криптографических преобразований.
  • Пароль пользователя нигде не сохраняется: если при вводе пароля не удается расшифровать настройки или файл ключей, будет выдано сообщение о том, что введен неверный пароль.

Чем проект может заинтересовать разработчиков?

Вы можете создать свой бизнес-автомат или интегрировать свои разработки с нашей программой и распространять на любых условиях (см. текст лицензии). Для сертификации вашего бизнес-автомата и получения бесплатной информационной поддержки, последуете нашим рекомендациям по созданию бизнес-автоматов.
